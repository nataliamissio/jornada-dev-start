# Exercício 4 — Refinamento sucessivo

## Nível 1: Visão Geral

1. Registrar os produtos.
2. Calcular o subtotal.
3. Verificar se o cliente tem desconto.
4. Mostrar o valor final.

## Nível 2: Detalhamento

### 1. Registrar os produtos
- Iniciar o sistema.
- Ler os produtos da compra.
- Guardar os valores dos produtos.

### 2. Calcular o subtotal
- Somar o valor de todos os produtos.
- Guardar o resultado na variável `subtotal`.

### 3. Verificar se o cliente tem desconto
- Perguntar se o cliente possui cartão fidelidade.
- Se tiver, aplicar 5% de desconto.
- Se não tiver, manter o valor do `subtotal`.

### 4. Mostrar o valor final
- Exibir o valor total a pagar.
- Finalizar a compra.
