## a. Calcular a área de um retângulo

```text
Início
  Leia base
  Leia altura
  area <- base * altura
  Escreva area
Fim
```

## b. Verificar se um número é par ou ímpar

```text
Início
  Leia numero
  Se (numero % 2 = 0)
    Escreva "par"
  Senão
    Escreva "ímpar"
Fim
```

## c. Encontrar o maior entre três números

```text
Início
  Leia A
  Leia B
  Leia C

  Se (A > B) e (A > C)
    Escreva A
  Senão se (B > A) e (B > C)
    Escreva B
  Senão
    Escreva C
Fim
```
