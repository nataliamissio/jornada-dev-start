# Exercício 1 — Função ou Procedimento?

**Str()** — **Função.** Retorna um número convertido para string.

**QOut()** — **Procedimento.** Apenas exibe uma mensagem na tela, sem retornar um valor.

**Date()** — **Função.** Retorna a data atual do sistema.

**Len()** — **Função.** Retorna a quantidade de caracteres de uma string ou de elementos de um array.

**Upper()** — **Função.** Retorna uma string convertida para letras maiúsculas.

**ClearScreen()** — **Procedimento.** Apenas limpa a tela, sem retornar nenhum valor.
